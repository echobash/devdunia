Base32 Encoder

Convert text and binary data to Base32 encoding format.

Original Size: 0 B
Base32 Size: 0 B
Overhead: 0%
Padding: 0 chars

Common Examples

Simple text:

"Hello World"

→ "JBSWY3DPEBLW64TMMQQQ===="

Binary data:

[0x48, 0x65, 0x6C, 0x6C, 0x6F]

→ "JBSWY3A="

JSON data:

{"name": "John", "age": 30}

→ "EJRGKIDROVUWG2ZAMJZGKYLDMU======"

File content:

Image files, PDFs, etc.

→ Long Base32 string
